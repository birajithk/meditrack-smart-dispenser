# Components List

## Required for Software + Embedded Prototype

| Component | Purpose |
|---|---|
| ESP32 DevKit | Main controller |
| Servo motor | Simulate compartment movement |
| IR sensor / break-beam sensor | Detect pill removal |
| Buzzer | User reminder |
| LED | Visual reminder |
| Resistors | LED protection |
| Breadboard | Temporary circuit |
| Jumper wires | Connections |
| 5V external power supply | Servo power |
| USB cable | ESP32 programming |

## Later Mechanical Prototype

| Component | Purpose |
|---|---|
| 3-4 compartment carousel | Store dummy pills |
| Fixed cover with access window | Prevent wrong compartment access |
| Motor bracket | Hold servo firmly |
| Enclosure | Hold full system |
| Sensor mount | Fix pill-removal sensor |
